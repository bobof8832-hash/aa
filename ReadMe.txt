# Care SelfBot Voice Joiner

## وصف المشروع

هذا سكربت سيلف بوت لحسابات ديسكورد العادية يتيح لك الانضمام تلقائياً إلى روم صوتي معين، مع إمكانية تفعيل كتم المايك (Mute) وتعطيل سماع الصوت (Deafen) عند الانضمام.

---

## المتطلبات

- Node.js (يفضل نسخة حديثة)
- مكتبات Node.js:
  - `discord.js-selfbot-v13`
  - `@discordjs/voice`

---

## طريقة التثبيت

1. افتح الطرفية في مجلد المشروع.  
2. نفذ الأمر التالي لتثبيت المكتبات:

```bash
npm install discord.js-selfbot-v13 @discordjs/voice
````

---

## إعداد الملف `careStore.js`

قم بإنشاء ملف باسم `careStore.js` وضع فيه بيانات حسابك والإعدادات المطلوبة:

```js
module.exports = {
  Token: 'YOUR_DISCORD_TOKEN',           // توكن حساب ديسكورد الخاص بك
  Guild: 'YOUR_GUILD_ID',                 // معرف السيرفر (Guild ID)
  Channel: 'YOUR_VOICE_CHANNEL_ID',      // معرف الروم الصوتي (Voice Channel ID)
  selfMute: true,                        // true لكتم المايك عند الانضمام، false لتشغيله
  selfDeaf: false                        // true لتعطيل سماع الصوت عند الانضمام، false لتفعيله
};
```

---

## كيفية التشغيل

لتشغيل السكربت، نفذ الأمر التالي في الطرفية:

```bash
node index.js
```

مع التأكد من أن اسم ملف السكربت هو `index.js` أو تغييره حسب اسم ملفك.

---

## ملاحظات هامة

* **استخدام سيلف بوت مخالف لشروط استخدام Discord، وقد يؤدي إلى حظر حسابك. استخدمه بحذر وعلى مسؤوليتك الخاصة.**
* تأكد من عدم مشاركة توكن حسابك مع أي شخص للحفاظ على أمان حسابك.
* يجب تفعيل وضع المطور (Developer Mode) في ديسكورد للحصول على معرفات السيرفر والروم.
* للتأكد من معرفات القنوات، انقر بزر الماوس الأيمن على الروم الصوتي واختر "نسخ المعرف" (Copy ID).
